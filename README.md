# Travique - Travel & Tour Website

Travique is a modern and fully responsive travel website designed to provide users with an engaging travel experience. The project features smooth animations, destination showcases, service sections, testimonials, and a clean user interface built using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Modern UI/UX
- ScrollReveal Animations
- Destination Showcase
- Services Section
- Testimonials Section
- Mobile-Friendly Navigation
- Smooth Scrolling
- Interactive Components

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- ScrollReveal.js
